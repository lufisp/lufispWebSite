Recommended Books

Docker

I've read the first 10 chapters out of a total of 13. This is a really nice book to start learning Docker. I highly recommend this one.

Big Data

I've started in Big Data with this Book. It's a very nice book that will give you a holistic view of some of the most popular tools in the hadoop ecosystem. Be brave, it's not an easy book to read! Especially for the newly arrived in this world.
It's a good book for learning Spark. If you work with the recent version, some examples won't work. Especially the ones in the Spark SQL chapter. For this topic I recommend the project's page.
Good book with a lot of examples to illustrate the concepts.
It's a nice introduction to HBASE. I read the "early edition" with only the first 7 chapters but it is more than the necessary to start constructing your Hbase system.
I'm still reading this one. So far so good, there is a lot of good examples to help us to see the power of this tool.

Java

I began learning JAVAEE with this book. It's fantastic! Don't be intimidated by the Introduction. All the concepts exposed there will be very well explained in the following chapters.
A nice book to have. There is some of the patterns presented in the GOF book, translated to the JavaEE world and others like Context Dependency Injection, asynchrony programming, Data object access with JPA...
I haven't finished this book yet but it certainly improved my code quality. This is a must have book!
A superb book of functional programming in java. After learning the power of the lambda expressions, you will never be the same :-)

Front-End

If you are looking for an introduction book for D3, this is the one! The author is very clear and precise. Unfortunately the version used is the third, that has a lot of incompatibility issues with the fourth version.
I read the second edition of this book a long time ago. Before that I used to do the style configuration in line. It's a good book, if you do all the examples that he propose you'll have a solid base in CSS. After that, if you have some programming skills, consider using Sass or Less to have still more power in working with styles.
This is a fantastic and complete JavaScript book. It will introduce you to some amazing piece of code. This book also has a chapter to introduce JQuery, SVG and WebGL. It's a must have book. But don't forget to learn ES2015, that comes with a lot of nice new functionalities like classes, promises, modules, and a lot more.
